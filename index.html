1.1 Języki formalne, pojęcia podstawowe

Alfabet (vocabulary) - dowolny, skończony, niepusty zbiór symboli

Alfabet - zbiór obiektów materialnych w języku matematyki. Mówimy o reprezentacji obiektów materialnych np. 5 - znak reprezentujący liczbę 5.

Przedmiotem zainteresowania automatów jest znak, który może być powielony w kilku miejscach

Definicja:
Słowem nad alfabetem V nazywamy dowolny skończony ciąg symboli z V (jest ono obiektem materialnym).

Najkrótsze słowo: ciąg pusty ε

a ∈ V jest przykładem najkrótszego 1-elementowego ciągu

Wszystkie słowa nad alfabetem V oznaczamy V *

V *\{ε} = V + - zbiór wszystkich niepustych słów nad alfabetem V.

Przykłady

  • V={a} - ε, a, aa, aaa ...
  • a 4 = aaaa - słowo złożone z 4 a
  • V = {0, 1} - wszystkie ciągi binarne są zbiorem nad alfabetem V
  • V = {0, 1, ... , 9} - słowa będące liczbą (ciąg cyfr)
  • V = {a, ą, b, c, ć, ... ,ź} - każdy ciąg składający się z tych znakó jest słowem, np. miasto, abżubr.

Celem gramatyki jest wskazywanie, które słowa nad zadanym alfabetem są uważane za poprawnie zbudowane

Definicja:
Językiem nad alfabetem V nazywamy dowolny podzbiór L zbioru V *

L ∈ V *

Przykład:
Językiem nad alfabetem binarnym będzie zbiór tych wszystkich ciągów, które mają długość mniejszą od 3
{ε, 0, 1, 00, 01, 10, 11}

1.2 Konkatenacja słów

Definicja:
Dla dowolnych słów ∈ P, Q V* konkatenacja P i Q jest słowem oznaczonym przez PQ i obliczanym w następujący sposób:
  1. P = a1, ..., an, n >= 1
    Q = b1, ..., bn, m >= 1
    PQ = a1, ..., anb1, ..., bn
  2. jeśli P = ε, to PQ = Q
  3. εε = ε

Konkatenacja jest działaniem łącznym:
(PQ)R = P(QR)

Ciąg pusty ε jest elementem neutralnym konkatenacji.

Konkatenacja nie tworzy grupy, ponieważ nie posiada elementó odwrotnych, ale jest półgrupą z jedynką.. Przemienność zachodzi tylko dla alfabetu złożonego z jednego znaku

P = Q 1QQ 2
Q 1 - prefix P
Q - infix P
Q 2 - sufix P

ε jest zawsze prefixem, infiksem i sufiksem w każdym słowie.

Q jest podsłowem słowa P, wtedy i tylko wtedy, gdy P = Q 1QQ 2 dla pewnych Q 1 Q 2

Definicja (długość słowa):
Długością słowa P ∈ V * nazywamy liczbę naturalną |p| zdefiniowaną indukcyjnie w następujący sposób:

  1. |ε| = 0
  2. |Pa| = |P| + 1

Długość słowa to liczba wystąpień symboli, które w nim są.

Własność:
|PQ| = |P| + |Q|

Dowód (indukcja po |Q|):

  1. |Q| = 0 (Q = ε)
  2. L = |PQ| = |Pε| = |P| = |P| + 0 = |P| + ε = |P| + |Q| = P
  3. Załóżmy, że |PQ| = |P| + |Q|, |Q|<= n
    L = |P(Qa)| = |(PQ)a| = |PQ| +1 = (|P| + |Q|) + 1 = |P| + (|Q| + 1) = |P| + |Qa| = P

Definicja (potęga słowa nad P ∈ V *):
niech P ∈ V *, n>= 0 , n-tą potęgą słowa P nazywamy słowo P n zdefiniowane indukcyjnie w najstępujący sposób:

  1. P 0 = ε
  2. P n + 1 = P nP

Definicja (odbicie zwierciadlane słowa P):
Niech P ∈ V *, odbiciem zwierciadlanym (rewersem) słowa P nazywamy słowo oznaczone przez P-1 definiowane indukcyjnie w następujący sposób:

  1. ε-1 = ε
  2. (Pa)-1 = aP-1

Przykład:
(abc) -1 = c(ab)-1 = cb(εa)-1 = cbaε = cba

Własności:

  1. (PQ)-1 = Q-1P-1
  2. (Pn)-1 = (P-1)n
  3. (P-1)-1 = P

Jeśli P = P-1 to słowo jest palindromem.

Sposoby definiowania języków

  1. Wymienienie elementów języka:
    L = {a, ab, bab} o ile L jest skończony L1 = {anbm, m, n >= 1} - L nieskończony
  2. Podanie pewnej własności strukturalnej wspólnej dla cech danego języka
    L2 = {anbn n>= 1}
    L3 = {P ∈ {a, b}* : #aP = #bP } #aP - liczba wystąpień a w słowie P
  3. Definicja indukcjna
  4. Podanie metody wyprowadzenia "słowa" (gramatyki)
    Reguła przepisywania:
    S -> AB
    A -> CD
    B -> AB

    S -> AB -> CDB -> CDAB -> CDAAB

    L2 = {anbn n>= 1}
    S -> ab
    S -> aSb

2.1 Deterministyczne automaty skończenie stanowe

q - głowica
T - skończony alfabet
K - zbiór stanów (skończony)
δ - faza przejścia automatu
q0 ∈ K - stan początkowy
H ∈ K - zbiór stanów akceptujących (kończących)

Definicja: Deterministycznym automatem skończenie stnowym nazywamy uporządkowaną piątkę A = ≺K, T, δ, q0, H>, gdzie:
T - skończony, niepusty alfabet taśmy (alfabet wejścia)
K - skończony, niepusty zbiór stanów
δ - faza przejścia automatu
q0 ∈ K - stan początkowy
H ∈ K - zbiór stanów końcowych (akceptujących)
δ K x T -> K

Definicja:
Rozszerzoną funkcją przejścia deterministycznego automatu skończenie stanowego A = ≺K, T, δ, q0, H> nazywamy funkcję δ ^ : K x T* zdefiniowaną indukcyjnie w następujący sposób:

  1. δ^(q, ε) = q
  2. δ^(q, Pa) = δ(δ^(q, P), a)

L(A) = { P ∈ T*: δ^(q0, P) ∈ H }

Twierdzenie:
δ^(q, PQ) = δ^(δ^(q, P), Q)

3.1 Niedeterministyczne automaty skończenie stanowe

Definicja: Niedeterministycznym automatem skończenie stnowym nazywamy uporządkowaną piątkę A = ≺K, T, δ, q0, H>, gdzie:
T - skończony, niepusty alfabet taśmy (alfabet wejścia)
K - skończony, niepusty zbiór stanów
δ - faza przejścia automatu
Q0 ∈ K - zbiór stanów początkowych
H ∈ K - zbiór stanów końcowych (akceptujących)
δ K x T -> P(K)
